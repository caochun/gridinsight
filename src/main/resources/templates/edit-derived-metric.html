计算公式

使用指标标识符引用其他指标，支持基本运算和数学函数
公式语法说明
基本运算：
  • + 加法
  • - 减法
  • * 乘法
  • / 除法
  • () 括号优先级
数学函数：
  • sqrt(值) 平方根
  • abs(值) 绝对值
  • max(值1,值2) 最大值
  • min(值1,值2) 最小值
示例： (指标A + 指标B) * 0.5 + sqrt(指标C)
