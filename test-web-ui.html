🚀 快速测试API

测试指标查询API：

curl "http://localhost:8080/api/metrics/query?identifier=中压拓扑.配变统计.配变总数"

测试指标列表API：

curl "http://localhost:8080/api/metrics/list"

测试批量查询API：

curl -X POST "http://localhost:8080/api/metrics/batch-query" \ -H "Content-Type: application/json" \ -d '["中压拓扑.配变统计.配变总数", "中压拓扑.拓扑关系准确率.拓扑关系准确率"]'

🛠️ 技术特性

  • 配置存储： 使用YAML文件存储指标定义，无需数据库
  • Web界面： 基于Bootstrap 5的现代化响应式UI
  • 公式引擎： 支持复杂数学表达式和指标依赖关系
  • 数据源： 支持HTTP API、MQTT、数据库、文件等多种数据源
  • 缓存机制： 内置指标值缓存，提升查询性能
  • RESTful API： 完整的REST API支持
